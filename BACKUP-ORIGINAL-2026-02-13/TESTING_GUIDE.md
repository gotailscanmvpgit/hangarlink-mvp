# HangarLink Features Testing Guide

## 1. HangarMatch AI Priority
- Log in as the admin user (`admin@hangarlink.com` / `password`).
- The admin account is pre-configured with alert preferences for `KJFK` and max price `$2000`.
- Go to `/matches` (via navigation "Matches" link if available, or direct URL).
- You will see listings scored by AI. Listings at KJFK will carry a "90%+ Match" badge.

## 2. Virtual Hangar Walkthrough
- As the admin (or any owner), go to "Rent Out Hangar" (`/post-listing`).
- Fill in the details.
- In the **360° Virtual Tour URL** field, paste a demo link (e.g., `https://matterport.com/discover/space/example`).
- Submit the listing.
- Go to the listing detail page.
- Observe the **"360° Virtual Tour"** button overlaid on the main photo. Click it to open the tour.

## 3. HangarLink Concierge
- Navigate to any listing detail page.
- Look for the **"Ask AI Concierge"** floating button at the bottom right.
- Click it to open the chat window.
- Type a query like "Is this hangar available?" or "What is the price?".
- The AI will respond with context-aware answers.

## 4. Seasonal Forecast & Alerts
- Go to the Homepage (`/`).
- Scroll down slightly below the hero section.
- Observe the **"Market Intelligence"** widget.
- It displays a simulated forecast (e.g., "Rising 20% Demand").

## 5. HangarLink Rewards
- Navigate to `/rewards`.
- Check your points balance. The admin account starts with 1,000 pts.
- Try to "Redeem" a reward (logic is simulated).

## 6. Pilot Co-Pilot Network
- Navigate to `/referrals`.
- You will see a unique referral code generated for your account.
- Click "Copy Code" to simulate sharing.

## 7. HangarLink Certified Program
- The Admin user is already "Certified".
- Post a listing as the Admin.
- View the listing in `/matches` or the main feed.
- It will display a **"Certified"** badge, increasing trust and visibility.
- To certify yourself as a normal user, visit `/admin/certify/me` (demo route).
